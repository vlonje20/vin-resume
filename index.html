Containerization

Containers are often described as “lightweight,” sharing the machine’s operating system kernel, unlike virtual machines that require a full operating system for each instance.

Orchestration

Cloud orchestration is the coordination, arrangement, or end-to-end automation of the deployment of services in a cloud-based environment. It introduces and enforces a workflow for automated activities of various processes to deliver the desired service to its client

Infrastructure as Code

Infrastructure as Code (IaC) is a method of managing and provisioning computing infrastructure (such as servers, networks, and storage) through machine-readable script files, rather than using manual processes.

Configuration

A configuration tool is a type of software that enables users to manage the configuration settings of various systems and applications, ensuring consistency, compliance, and efficient maintenance.

Continuous Integration/Delivery

Continuous Integration/Continuous Delivery (CI/CD) is a software development practice that involves automating the processes of code integration, testing, deployment, and delivery.

Monitoring

Monitoring is a critical aspect of IT operations and software development that involves the systematic observation and measurement of a system's performance, availability, and resource utilization.
